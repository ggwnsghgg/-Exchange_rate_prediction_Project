# Exchange_rate_prediction_Project



플레이데이터 에서 진행한
</br>
달러, 한화, LIBOR, Swap Rates 데이터를 이용하여 LSTM과 앙상블(xgboost)를 이용하여 환율을 예측한 프로젝트입니다.
</br>
완성하지는 못한 프로젝트이지만 유의미한 결과가 나왔다고 생각하여 프로젝트를 따로 남겨놨습니다.
</br>
도움이 되셨으면 좋겠습니다.
